# React Soup is an ejected create-react-app with cool stuff added

- [Create-react-app ejected](https://github.com/facebookincubator/create-react-app)
- [Stylus-loader](https://github.com/shama/stylus-loader)
- [Milligram Stylus](https://github.com/milligram/milligram-stylus)
- [React-router](https://reacttraining.com/react-router/web/api/BrowserRouter)
- [React-router-redux](https://github.com/reacttraining/react-router/tree/master/packages/react-router-redux)
- [Redux](http://redux.js.org/docs)


## Audience

For react developers starting projects looking for a simple boilerplate or instructions on how to plumb stuff together.
